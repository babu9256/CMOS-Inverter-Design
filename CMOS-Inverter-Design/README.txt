# CMOS Inverter Design using Cadence Virtuoso

This project demonstrates the design and verification of a CMOS inverter using Cadence Virtuoso.

## Project Includes
- Schematic Design
- Layout Design
- DRC Verification
- LVS Verification

## Tools Used

- Assura / Calibre

## Results

- LVS Matched Successfully
